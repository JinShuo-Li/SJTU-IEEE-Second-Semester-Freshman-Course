## Pointers and Arrays

### What is Memory

Memory is a component of a computer that stores data and instructions. It is organized into a series of addresses, each of which can hold a value. In programming, we often use pointers to access and manipulate memory.

It's a series of bytes, each with a unique address. When we declare a variable, the computer allocates a specific amount of memory for that variable and assigns it an address. For example, if we declare an integer variable `int x;`, the computer might allocate 4 bytes of memory for `x` and assign it an address like `0x7ffee3b8c8f0`.

### L/R Values

In programming, an L-value (left value) refers to an object that has a memory address and can be assigned a value. An R-value (right value) refers to a value that does not have a memory address and cannot be assigned a value.
